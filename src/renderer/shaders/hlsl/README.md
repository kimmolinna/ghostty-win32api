# HLSL Shaders

HLSL shader source for the DirectX renderer.

## Compiling

Requires the DirectX Shader Compiler (`dxc.exe` from the Windows SDK or Vulkan SDK):

```
dxc -T vs_6_0 -E VSMain -Fo cell_vs.dxil cell.hlsl
dxc -T ps_6_0 -E PSMain -Fo cell_ps.dxil cell.hlsl
```

The `.dxil` files will be generated by the build system (HlslStep.zig) and
embedded at comptime via `@embedFile` in the pipeline module.

The `.dxil` files are not committed to git - they are build artifacts.
